HTML 5

HTML stands for hypertext markup language, and it is used to lay out the basic structure of a website. You can build a simple website with HTML alone, including different font sizes and types, paragraphs, photos, background colors and other basic features.
To make your site look modern and nicely organised, additional languages may be learnt. According to the learned skill, writing html code is basically writing instructions for a web browser on how to dispay a website. To write html code, all you have to do is understand the proper syntax for communicating with a computer.
